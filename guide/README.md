# 오늘은 제가 섬기겠습니다 사용법

대상 URL: https://haseong23.github.io/let-me-know-your-menu/?cell=demo

시크릿 모드처럼 처음 들어온 사용자를 기준으로 정리했습니다. 실제 공유 데모 데이터 변경을 피하기 위해 주문 제출과 마감은 누르지 않았고, 섬김 시작 화면은 같은 앱을 로컬 테스트 환경에서 재현해 캡처했습니다.

## 1. 내 이름을 고릅니다

링크에 들어오면 구성원 목록이 보입니다. 본인 이름을 누릅니다.

이름이 없으면 아래 입력칸에 이름을 적고 `추가하고 시작`을 누릅니다.

![이름 선택 화면](menu-polished/01-name-select.png)

## 2. 선택한 이름을 한 번 더 확인합니다

선택 표시가 붙으면 아래의 `... 님으로 시작하기` 버튼을 누릅니다. 이 단계는 다른 사람 이름으로 주문하는 실수를 줄이기 위한 확인 단계입니다.

![이름 확인 화면](menu-polished/02-confirm-name.png)

## 3. 오늘 주문을 엽니다

아직 아무도 주문을 열지 않았다면 `오늘은 제가 섬기겠습니다` 버튼이 보입니다. 이 버튼을 누르면 내가 오늘 섬기는 분으로 설정되고, 구성원들이 메뉴를 고를 수 있습니다.

이미 누군가 주문을 열어둔 상태라면 이 화면 없이 바로 메뉴 선택 화면으로 들어갑니다.

![섬김 시작 화면](menu-polished/03-host-start.png)

## 4. 메뉴를 고릅니다

희망하는 메뉴를 선택합니다. 원하는 메뉴가 없으면 `전체 메뉴판`을 눌러 찾습니다.

오늘 마시지 않는 사람은 `오늘은 안 마실게요`를 누르면 됩니다.

![메뉴 선택 화면](menu-polished/04-menu-open.png)

## 5. 옵션을 정하고 주문합니다

`ICE` 또는 `HOT`을 고르고 `이걸로 주문`을 누릅니다.

![옵션 선택 화면](menu-polished/05-options-submit.png)

## 6. 현황을 확인합니다

섬기는 분은 `현황`에서 총 잔수, 주문 완료, 안 마심, 미응답 인원을 확인합니다.

필요하면 미응답자 안내 문구를 복사하고, 주문이 끝나면 `주문 마감하기`를 누릅니다.

![주문 현황 화면](menu-polished/06-status.png)

## 한 줄 요약

이름 선택 → 이름 확인 → `오늘은 제가 섬기겠습니다` → 메뉴 선택 → 온도 선택 → `이걸로 주문` → 현황 확인
